### 個人照片
- `professor-photo.jpg` - 侯勝宗教授的個人照片（建議尺寸：400x400px 或更大）

### 服務影像（9張）
- `service-1.jpg` - 微光行動協會服務場景（老人與司機的互動）
- `service-2.jpg` - 協助輪椅使用者的場景
- `service-3.jpg` - 司機培訓與轉型的宣傳圖片
- `service-4.jpg` - 「交通不只是移動，而是生命的連結」宣傳圖片
- `service-5.jpg` - 傳善獎經驗分享會的照片
- `service-6.jpg` - 「多走一哩路」宣傳圖片（老人照片）
- `service-7.jpg` - 微光接送服務場景（老人與司機握手）
- `service-8.jpg` - 偏鄉交通服務場景（山路與車輛）
- `service-9.jpg` - 小驢行無障礙服務（輪椅使用者上車）

### 首頁背景（可選）
- `hero-background.jpg` - 首頁橫幅背景圖片（建議尺寸：1920x1080px）

## 圖片建議規格

- **格式**：JPG 或 PNG
- **品質**：高品質，但檔案大小控制在 500KB 以下（可使用圖片壓縮工具）
- **尺寸**：
  - 個人照片：至少 400x400px
  - 服務影像：至少 800x600px（4:3 比例）
  - 首頁背景：1920x1080px 或更大

## 注意事項

1. 所有圖片檔案名稱必須與上述名稱完全一致（區分大小寫）
2. 如果暫時沒有圖片，網站會顯示替代內容
3. 建議使用圖片壓縮工具（如 TinyPNG）來優化檔案大小，以提升網站載入速度
