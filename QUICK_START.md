## 常見問題 ❓

### Q1: 看不到新區塊？
**A**: 確保：
- 已刷新瀏覽器頁面（Ctrl+F5 或 Cmd+Shift+R）
- 檢查是否使用了快取的舊版本
- 確認 index.html 和 style.css 都已更新

### Q2: 語言切換沒反應？
**A**: 請檢查：
- 瀏覽器控制台是否有JavaScript錯誤
- main.js 檔案是否正確載入
- 瀏覽器是否啟用了JavaScript

### Q3: 手機上顯示不正常？
**A**: 建議：
- 在實際設備上測試，而非僅用瀏覽器模擬
- 檢查 viewport meta 標籤是否正確
- 清除瀏覽器快取後重試

### Q4: 動畫效果太慢或太快？
**A**: 這是正常的：
- 動畫時間設定為 0.3 秒
- 可在 CSS 中調整 `--transition-base` 變數
- 如需無動畫，瀏覽器的「減少動畫」設定會自動處理
