# 侯勝宗教授個人網站

這是侯勝宗教授的個人網站，展示其在社會創新、服務創新、移動服務等領域的專業成就與社會貢獻。

## 最新更新 (2024-12-02)

### 新增「自我成長領導力」專區

為網站新增了一個專門的「自我成長領導力」(Self-Growth Leadership) 區塊，內容包括：

#### 主要內容模塊：

1. **領導力哲學** - 四大核心理念：
   - 自我覺察與反思
   - 目標導向與使命感
   - 同理心與服務精神
   - 終身學習與成長

2. **領導力成長框架** - 六步成長路徑：
   - 自我認識
   - 願景塑造
   - 能力建構
   - 關係經營
   - 實踐行動
   - 持續迭代

3. **學習資源** - 四種學習途徑：
   - 工作坊與課程
   - 讀書會與共學
   - 一對一輔導
   - 實踐專案

#### 設計特色：

- **多語言支持**：完整支援繁體中文、英文、日文三種語言
- **響應式設計**：適配桌面、平板、手機等各種設備
- **視覺層次**：使用漸層色彩、卡片設計、陰影效果等現代化UI元素
- **互動體驗**：hover動畫、平滑滾動、過渡效果等提升用戶體驗
- **配色方案**：沿用網站既有的專業藍色系搭配溫暖橙色/米色

#### 技術實現：

- 在 `index.html` 中添加完整的領導力區塊結構
- 在 `css/style.css` 中添加對應的樣式定義
- 在導航欄中添加「自我成長領導力」連結
- 確保與現有功能（語言切換、平滑滾動、響應式選單）完全整合

## 網站結構

```

├── index.html              # 主頁面
├── css/
│   └── style.css          # 樣式表
├── js/
│   ├── main.js            # 主要JavaScript功能

```

## 主要功能

- **多語言切換**：支援繁體中文、英文、日文
- **響應式設計**：完美適配各種螢幕尺寸
- **平滑滾動導航**：流暢的頁面內導航體驗
- **表單驗證**：即時表單驗證與錯誤提示
- **滾動動畫**：元素進入視窗時的優雅動畫效果
