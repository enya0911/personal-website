## 🎨 設計特色

### 視覺設計
- **漸層背景**：淡藍色 → 米色 → 白色的流暢過渡
- **卡片設計**：圓角、陰影、漸層背景的現代化卡片
- **配色方案**：
  - 主色：專業藍色 (#2c5f8d)
  - 強調色：溫暖橙色 (#d4a574)
  - 背景：淡藍色 (#e8f0f7) 和米色 (#f5f1eb)

### 互動效果
- **懸停動畫**：卡片向上浮動 + 陰影加深
- **過渡效果**：流暢的0.3秒過渡動畫
- **滾動動畫**：內容進入視窗時的淡入向上效果
- **框架項目**：左側色條漸顯動畫

### 排版設計
- **網格佈局**：自適應的響應式網格
- **層次分明**：清晰的標題、副標題、正文結構
- **視覺焦點**：數字編號、圖標、引言的視覺突出

## 🚀 如何使用

### 立即預覽
```bash
# 方法1：直接在瀏覽器打開
open index.html

# 方法2：啟動本地伺服器
python3 -m http.server 8000
# 然後訪問 http://localhost:8000
```

### 查看新區塊
1. 打開網站
2. 點擊導航欄的「自我成長領導力」
3. 或直接訪問 `index.html#leadership`

### 測試語言切換
1. 點擊右上角的語言切換按鈕
2. 觀察所有內容的語言變化
3. 測試順序：中文 → 英文 → 日文 → 中文
